An `AutonomousMode` is a pre-programmed sequence of actions that your `Robot` will perform autonomously.

## Guide

An `AutonomousMode` defines a sequence of `Action`s for your `Robot` to execute.

### 1. Create Your `AutonomousMode` Class

First, create a new class that inherits `AutonomousMode`.
The constructor requires a `robotFactory`, which is used to create your `Robot` instance.
Use a lambda function to provide the `robotFactory`.

<Tabs>

<TabItem label="Kotlin">

```kotlin
import dev.kingssack.volt.opmode.autonomous.AutonomousMode

class ExampleAuto : AutonomousMode<MyRobot>(robotFactory = { hardwareMap -> MyRobot(hardwareMap) }) {
    override fun sequence() = execute {
        // Your actions will go here
    }
}
```

</TabItem>

<TabItem label="Java">

```java
import dev.kingssack.volt.opmode.autonomous.AutonomousMode;

public class ExampleAuto extends AutonomousMode<MyRobot> {
    public ExampleAuto() {
        super(MyRobot::new);
    }

    @Override
    public void sequence() {
        execute(() -> {
            // Your actions will go here
        });
    }
}
```

</TabItem>

### 2. Make A Sequence of `Action`s

We'll open the claw, then close the claw and score at the same time. We'll repeat this sequence three times.

<Tabs>

<TabItem label="Kotlin">

```kotlin {5-12}
import dev.kingssack.volt.opmode.autonomous.AutonomousMode

class ExampleAuto : AutonomousMode<MyRobot>(robotFactory = { hardwareMap -> MyRobot(hardwareMap) }) {
    override fun sequence() = execute {
        repeat(3)  {
            +robot.claw.open() // Add an action to the sequence

            parallel { // Run these actions at the same time
                +robot.claw.close()
                +robot.score()
            }
        }
    }
}
```

</TabItem>

<TabItem label="Java">

```java {11-18}
import dev.kingssack.volt.opmode.autonomous.AutonomousMode;

public class ExampleAuto extends AutonomousMode<MyRobot> {
    public ExampleAuto() {
        super(MyRobot::new);
    }

    @Override
    public void sequence() {
        execute(builder -> {
            for (int i = 0; i < 3; i++) {
                builder.unaryPlus(robot.getClaw().open()); // Add an action to the sequence

                parallel(() -> { // Run these actions at the same time
                    builder.unaryPlus(robot.getClaw().close());
                    builder.unaryPlus(robot.score());
                });
            }
        });
    }
}
```

</TabItem>

### 3. Make it an OpMode

Add the `@Autonomous` annotation to the class definition. Without this annotation, the OpMode will not be registered and will not appear on the Driver Station.

<Tabs>

<TabItem label="Kotlin">

```kotlin {4}
import com.qualcomm.robotcore.eventloop.opmode.Autonomous
import dev.kingssack.volt.opmode.autonomous.AutonomousMode

@Autonomous(name = "Example Auto", group = "Examples")
class ExampleAuto : AutonomousMode<MyRobot>(robotFactory = { hardwareMap -> MyRobot(hardwareMap) }) {
    override fun sequence() = execute {
        repeat(3)  {
            +robot.claw.open()

            parallel {
                +robot.claw.close()
                +robot.score()
            }
        }
    }
}
```

</TabItem>

<TabItem label="Java">
