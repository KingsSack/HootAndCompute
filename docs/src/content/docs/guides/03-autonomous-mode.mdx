An `AutonomousMode` is a pre-programmed sequence of actions that your `Robot` will perform autonomously.

## Guide

This guide will walk you through the process of setting up your own `AutonomousMode`.

### 1. Create Your `AutonomousMode` Class

First, create a new class that inherits `AutonomousMode`.
implement the `getRobot` method, which is used to create your `Robot` instance.
override the name field, which is the name that will show up on the driver hub
make sure it has a no-parameter constructor.
your opmodes will automatically be registered, you don't have to worry about annotations.

<Tabs>

<TabItem label="Kotlin">

```kotlin
import dev.kingssack.volt.opmode.autonomous.AutonomousMode
import com.qualcomm.robotcore.hardware.HardwareMap

class ExampleAuto : AutonomousMode<MyRobot>() {
    override val name = "example auto"
    override val group = "examples"
    override val autoTransition = "example teleop"
    override fun getRobot(hardwareMap: HardwareMap) {
        return MyRobot(hardwareMap)
    }
    override fun sequence() = execute {
        // Your actions will go here
    }
}
```

### 2. Make A Sequence of `Action`s

We'll open the claw, then close the claw and score at the same time. We'll repeat this sequence three times.

<Tabs>

<TabItem label="Kotlin">

```kotlin {5-12}
import dev.kingssack.volt.opmode.autonomous.AutonomousMode
import com.qualcomm.robotcore.hardware.HardwareMap

class ExampleAuto : AutonomousMode<MyRobot>() {
    override fun getRobot(hardwareMap: HardwareMap) : MyRobot {
        return MyRobot(hardwareMap)
    }
    override fun sequence() = execute {
        repeat(3)  {
            +robot.claw.open() // Add an action to the sequence

            parallel { // Run these actions at the same time
                +robot.claw.close()
                +robot.score()
            }
        }
    }
}
```

</TabItem>

<TabItem label="Java">

```java {11-18}
import dev.kingssack.volt.opmode.autonomous.AutonomousMode;

public class ExampleAuto extends AutonomousMode<MyRobot> {
    @Override
    public MyRobot getRobot(HardwareMap hardwareMap) {
        return new MyRobot(hardwareMap);
    }

    public ExampleAuto() {
        super();
    }

    @Override
    public String getName() {
        return "example auto";
    }
    @Override
    public String getGroup() {
        return "examples";
    }
    // optional auto transition
    @Override
    public String getAutoTransition() {
        return "example teleop";
    }

    @Override
    public void sequence() {
        execute(builder -> {
            for (int i = 0; i < 3; i++) {
                builder.unaryPlus(robot.getClaw().open()); // Add an action to the sequence

                parallel(() -> { // Run these actions at the same time
                    builder.unaryPlus(robot.getClaw().close());
                    builder.unaryPlus(robot.score());
                });
            }
        });
    }
}
```
