## What is a [Robot](/api/Volt/-volt/com.lasteditguild.volt.robot/-robot)?

A [Robot](/api/Volt/-volt/com.lasteditguild.volt.robot/-robot) is a class that contains all of the [Attachments](kdoc/volt/-volt/com.lasteditguild.volt.attachment/-attachment) and some [Actions](kdoc/volt/-volt/com.lasteditguild.volt.action/-action) that make up a robot.

You should create a new [Robot](/api/Volt/-volt/com.lasteditguild.volt.robot/-robot) class for each physical robot you have built.

## Creating a [Robot](/api/Volt/-volt/com.lasteditguild.volt.robot/-robot)

Create a new kotlin class in `TeamCode/src/main/kotlin/org/firstinspires/ftc/teamcode/robot` that inherits the [Robot](/api/Volt/-volt/com.lasteditguild.volt.robot/-robot) class.

The [Robot](/api/Volt/-volt/com.lasteditguild.volt.robot/-robot) class contains:

1. A list of attachments
2. A drive class
3. A function called `update(telemetry: Telemetry)` that is called every tick

### Add Sensors

Define the sensors that your robot will use and a way to use them. For example, add a HuskyLens sensor to your robot.

<Tabs>

<TabItem label="Kotlin">

```kotlin {2-3, 6-7, 10-18}
class Example(hardwareMap: HardwareMap, initialPose: Pose2d) : Robot(hardwareMap, initialPose) {
    override val drive = SimpleMecanumDrive(hardwareMap, initialPose)

    // Sensors
    private val huskyLens: HuskyLens = hardwareMap.get(HuskyLens::class.java, "lens")

    init {
        // Set huskylens mode
        huskyLens.selectAlgorithm(HuskyLens.Algorithm.OBJECT_RECOGNITION)
    }

    fun getDetectedObjects(telemetry: Telemetry): Array<out HuskyLens.Block>? {
        // Get objects
        val blocks = huskyLens.blocks()
        telemetry.addData("Block count", blocks.size)
        for (block in blocks) {
            telemetry.addData("Block", block.toString())
        }
        return blocks
    }
}
```

### Add Actions

Define actions for your attachments and sensors. For example, add a function to move a motor to a specific position.

<Tabs>

<TabItem label="Kotlin">

```kotlin {15-18}
class Example(hardwareMap: HardwareMap, initialPose: Pose2d) : Robot(hardwareMap, initialPose) {
    override val drive = SimpleMecanumDrive(hardwareMap, initialPose)

    // Sensors
    private val huskyLens: HuskyLens = hardwareMap.get(HuskyLens::class.java, "lens")

    // Attachments
    val exampleAttachment: SimpleAttachmentWithDcMotor = SimpleAttachmentWithDcMotor(hardwareMap, "motor")

    init {
        attachments = listOf(exampleAttachment)

        // Set huskylens mode
        huskyLens.selectAlgorithm(HuskyLens.Algorithm.OBJECT_RECOGNITION)
    }

    fun moveMotorToSpecificPosition(): Action {
        // Move motor to 1000 at 50% power
        return exampleAttachment.goTo(0.5, 1000, 0, 1000)
    }

    fun getDetectedObjects(telemetry: Telemetry): Array<out HuskyLens.Block>? {
        // Get objects
        val blocks = huskyLens.blocks()
        telemetry.addData("Block count", blocks.size)
        for (block in blocks) {
            telemetry.addData("Block", block.toString())
        }
        return blocks
    }
}
```
