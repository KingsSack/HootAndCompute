---
title: Using Samples
---

A great way to learn more about coding for First Tech Challenge (FTC) is to use the samples
located in the `external.samples` package in the [FtcRobotController](https://github.com/FIRST-Tech-Challenge/FtcRobotController) module.

Here you can find Java samples for almost every single actuator and sensor, which can be helpful when programming [Attachments](../../guides/02-attachments) in Volt.
